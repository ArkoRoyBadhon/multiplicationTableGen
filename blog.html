JavaScript Questions

1. What is the difference between let, const and var in JavaScript?

==> let, const, var all are the variable declaration keywords in JavaScript. Below i am discuss the key features of these declaration keywords -

Var: "var" is function scoped. "var" can be reassigned and redeclared.
Both let and const are block scoped.

Let: "let" a block scoped variable. "let" can be reassigned but cannot be redeclared at the same scope.

Const: "const" is also a block scoped variable. "const" cannot be reassigned but can be redeclared at the same scope.


2. What is the difference between find(), filter(), method?

==> find() and filter() are both introduced in ES6. we can use them in our code in different purposes. I give some idea below -

find(): "find()" method returns the first element of an array that satisfies the given condition. If this method finds its first match then the method close the iteration. It means if the given array is too long but it finds a matched value then it not check the rest of the array.

filter(): "filter()" method returns an array of elements that satisfy the condition. We also need an array to perform this method. If the array is longer but the exprected result found at a early index still it checks the whole array and returns the outpur array.

If we need to check one element then we should use find() method and if we need a collection of elements then we should use filter() method. for one element find() method is more faster than filter() method.
